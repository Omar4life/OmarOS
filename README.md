# OmarOS

A web-based operating system simulator built to showcase projects, tech blogs, and professional profiles in a clean retro interface.

## 🚀 Features

* **Retro UI Design** – Framed interface mimicking classic system aesthetics.
* **Live System Clock** – Real-time dynamic date and time tracking.
* **Integrated Social Hub** – Quick-access structural links for professional networking.
* **Lightweight Architecture** – Built with core web standards for zero-latency loading.

## 🛠️ Built With

* **HTML5** – Content structure.
* **CSS3** – Visual styling.
* **JavaScript** – Dynamic time logic.

## 💻 How to Run It Locally

```bash
git clone https://github.com
cd omaros
open index.html
```
