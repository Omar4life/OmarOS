# OmarOS

This is my very own WebOS, which is called OmarOS (I mean obviously, what else would I call it), OmarOS is a WebOS which is basically everything about me

## 🚀 Features

* **Retro UI Design** – Framed interface mimicking classic system aesthetics.
* **Live System Clock** – Real-time dynamic date and time tracking.
* **Integrated Social Hub** – Quick-access structural links for professional networking.
* **Lightweight Architecture** – Built with core web standards for zero-latency loading.

## 🛠️ Built With

* **HTML** – Base of the app.
* **CSS** – Some cool drip for the app.
* **JavaScript** – Everyone needs to know the time, right?
